# Book Search Engine

## Description
This is a MERN Stack project for searching books from www.googleapis.com/books.

## Technologies Used
- HTML
- CSS
- JavaScript (ES6)
- React
- React Bootstrap
- React Router
- MongoDB
- GraphQL

## Usage
Login to your account, or register an account. Search for books, and save them to see later in the "See Your books" section.
